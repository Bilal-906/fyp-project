## üß© Detailed Feature Descriptions

### üßÆ **f1 ‚Äì f200: Numeric Network Flow Features**

These columns represent quantitative characteristics derived from raw network traffic.
Each row corresponds to a single **network connection (flow)**.
Although the data is synthetic, it mimics real-world intrusion detection features inspired by datasets like **KDD Cup 99**, **NSL-KDD**, and **CIC-IDS2017**.

| **Range**       | **Feature Type**                        | **Description**                                                              | **Example Real-World Analogy**                                 |
| --------------- | --------------------------------------- | ---------------------------------------------------------------------------- | -------------------------------------------------------------- |
| **f1 ‚Äì f20**    | Flow Volume Metrics                     | Measures packet and byte counts in both directions.                          | `total_bytes_sent`, `total_bytes_received`, `packet_count`     |
| **f21 ‚Äì f40**   | Time & Duration Metrics                 | Represents how long a connection lasted and how packets were spaced in time. | `flow_duration`, `avg_interarrival_time`, `active_time`        |
| **f41 ‚Äì f60**   | Rate Metrics                            | Speed of data transfer or packet emission rate.                              | `bytes_per_sec`, `packets_per_sec`, `throughput`               |
| **f61 ‚Äì f80**   | Header-Based Metrics                    | Statistics extracted from IP/TCP headers.                                    | `header_size_avg`, `flag_count`, `tcp_window_size`             |
| **f81 ‚Äì f100**  | Statistical Variation                   | Variance, skewness, or entropy in flow statistics.                           | `packet_size_std`, `entropy_src_ip`, `flow_variance`           |
| **f101 ‚Äì f120** | Content-Based Indicators                | Reflects patterns in packet payload or connection contents.                  | `malicious_keyword_score`, `payload_entropy`, `content_length` |
| **f121 ‚Äì f140** | Connection Behavior Features            | Derived from counts of repeated, failed, or reset connections.               | `failed_logins`, `connection_resets`, `duplicate_flags`        |
| **f141 ‚Äì f160** | Host-Based Aggregates                   | Average behavior per host or per session.                                    | `avg_packets_host`, `avg_bytes_per_connection`                 |
| **f161 ‚Äì f180** | Statistical Summaries                   | Derived features summarizing overall session distributions.                  | `mean_flow_rate`, `std_dev_packet_count`                       |
| **f181 ‚Äì f200** | Randomized Noise & Low-Variance Metrics | Minor or non-informative attributes to add realism and prevent overfitting.  | Dummy or uncorrelated features                                 |

All numeric features have been standardized to **zero mean and unit variance**, making them suitable for ML models without additional normalization.
